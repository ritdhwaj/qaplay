Network Traffic Capture

Real-time monitoring of all HTTP requests and responses with support for Chromium, Firefox, and Safari browsers.

  • Filter by origin to focus on specific domains
  • View complete request/response details
  • Status-based color coding (2xx, 3xx, 4xx, 5xx)
  • Headers, body, and timing information

API Mocking

Create mocks directly from captured requests. Configure custom responses with status codes, headers, and delays.

  • One-click mock creation from captured requests
  • Custom status codes, headers & response delays
  • Toggle mocks on/off without deleting
  • Import/Export mocks as JSON for team sharing

Session Recording

Automatic recording of all user interactions with Playwright script generation following Page Object Model structure.

  • Capture clicks, inputs, and navigation
  • Generate Playwright test scripts (POM)
  • Built-in code editor for script editing
  • Run tests directly with HTML reports

Performance Analytics

Lighthouse-powered audits measuring Performance, Accessibility, Best Practices, SEO, and PWA scores.

  • Google Lighthouse integration
  • Auto-analytics for all visited pages
  • Detailed HTML reports
  • Color-coded scores for quick assessment

Error Tracking

Dedicated tab for filtering requests with 4xx/5xx status codes for quick identification of failing API calls.

  • Automatic filtering of error responses
  • Quick identification of failing APIs
  • Create mocks from error responses
  • Test error handling scenarios

Code Snippets Library

Pre-built Playwright code snippets for common assertions and actions to speed up your script creation.

  • Assertions (visibility, text, enabled state)
  • Actions (clicks, fills, navigation)
  • Waiting and timeout helpers
  • One-click insertion into scripts
