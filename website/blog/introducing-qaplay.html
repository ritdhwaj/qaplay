The world of QA testing has evolved dramatically over the past few years. Modern web applications are more complex than ever, with intricate API interactions, real-time data flows, and sophisticated user interfaces. Yet, QA engineers and developers often find themselves juggling multiple tools to capture network traffic, mock APIs, record sessions, and generate test scripts.

That's where QAPlay comes in.

The Problem We're Solving

As a QA engineer, your typical workflow might look like this:

  • Use Chrome DevTools to inspect network requests
  • Switch to a separate tool like Postman for API mocking
  • Manually write Playwright or Selenium test scripts
  • Use Lighthouse CLI for performance audits
  • Coordinate all these tools while trying to maintain focus

This context-switching is not just annoying‚Äîit's expensive. It slows down your testing workflow, increases cognitive load, and makes it harder to spot patterns across different aspects of your application's behavior.

üí° The QAPlay Solution: A single, unified desktop application that brings together network capture, API mocking, session recording, test generation, and performance analytics in one seamless interface.

Key Features That Make a Difference

1. Real-Time Network Traffic Capture

QAPlay monitors all HTTP requests and responses as you browse, giving you complete visibility into your application's network behavior. Filter by domain, inspect headers and bodies, and export data for further analysis‚Äîall in real-time.

2. One-Click API Mocking

See an interesting API request? Turn it into a mock with a single click. Customize the response body, status code, headers, and even add artificial delays to simulate slow networks or test timeout scenarios. Your mocks are applied automatically in subsequent browser sessions.

3. Automatic Test Script Generation

QAPlay records your browser interactions and generates Playwright test scripts following the Page Object Model (POM) pattern. Edit the generated code in the built-in editor, add assertions from our code snippets library, and run tests directly from the app.

4. Lighthouse-Powered Performance Analytics

Get comprehensive performance, accessibility, SEO, and PWA scores powered by Google Lighthouse. Enable auto-analytics mode to audit every page you visit during a testing session, making performance testing effortless.

5. Smart Error Tracking

A dedicated Errors tab automatically filters all 4xx and 5xx responses, making it easy to identify failing API calls and create mocks for testing error handling scenarios.

Built for Modern Development

QAPlay is built using cutting-edge technologies:

  • Electron 27 - Cross-platform desktop framework
  • React 18 - Modern, responsive UI
  • Playwright 1.40 - Powerful browser automation
  • Lighthouse 11 - Industry-standard performance auditing

This technology stack ensures QAPlay is fast, reliable, and supports Chromium, Firefox, and WebKit browsers out of the box.

Designed for Your Workflow

We've put a lot of thought into making QAPlay intuitive and efficient. The interface uses modern design principles with color-coded network requests, expandable detail views, and smart defaults that just work. Mocks can be toggled on/off without deletion, scripts can be batch-executed, and performance reports are saved automatically for comparison.

Free and Open Source

QAPlay is completely free to download and use. We believe in providing value to the QA community and fostering an ecosystem where quality is easier to achieve for everyone. Whether you're a solo developer, part of a QA team, or contributing to open source projects, QAPlay is designed to make your testing workflow faster and more enjoyable.
